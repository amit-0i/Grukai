## Installation

For hyprdots (old)
```sh
Hyde theme import "Grukai" "https://github.com/amit-0i/Grukai"
```

For HyDE (new)
```sh
hydectl theme import
```

and choose Grukai
---

or install another theme from [HyDe Gallery](https://github.com/kRHYME7/hyde-gallery)
